## Run Locally

Clone the project

```bash
  git clone https://github.com/shriram083/powerful-nut-6425.git
```

Go to the project directory

```bash
  cd powerful-nut-6425
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the localhost server

```bash
  npm start
```
